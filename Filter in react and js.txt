Filtering in React is pretty much what it says it is.
 It’s the process of looping through an array and including or excluding elements inside 
that array based on a condition that you provide.


Let’s assume that you want to only display names that include the letter ‘J’ in them.

import React from 'react';

const names = ['James', 'John', 'Paul', 'Ringo', 'George'];

function App() {
  return (
    <div>
      {names.filter(name => name.includes('J')).map(filteredName => (
        <li>
          {filteredName}
        </li>
      ))}
    </div>
  );
}

export default App;


******Did you know that you can chain array function in JavaScript, such as filter, map, 
and reduce? This can save many lines of code, 
although it may be difficult to understand for those new to coding. *****

Example 2: Filter an Array of Objects by Value in React

import React from 'react';

const people = [
  {
    name: 'James',
    age: 31,
  },
  {
    name: 'John',
    age: 45,
  },
  {
    name: 'Paul',
    age: 65,
  },
  {
    name: 'Ringo',
    age: 49,
  },
  {
    name: 'George',
    age: 34,
  }
];

function App() {
  return (
    <div>
      {people.filter(person => person.age < 60).map(filteredPerson => (
        <li>
          {filteredPerson.name}
        </li>
      ))}
    </div>
  );
}

export default App;
